This notebook will detail the creation of a single speaker playback chirp series. This set of files are for the single speaker ultrasonic playbacks. The sweeps are of course of much larger bandwidth, and are not constrained to follow a certain set of durations or time-frequency structure.

The single speaker Ultrasound playbacks are :¶

For the single-speaker playbacks I'd like to include the following types of playbacks in the mp3 recordings:

  1. Linear downward sweep
  2. Logarithmic downward sweep
  3. Linear bidirectional chirp

All of the above chirps will be played in the following durations:

  1. 0.024 ms
  2. 0.012 ms
  3. 0.006 ms

This essentially corresponds to a set of 9 calls in a series, each of a particular waveform type and duration. One set of nine chirps with the 200 ms silences will be : 0.2 * 9 = 1.8 seconds long. This will be the repeating unit for all the chirp series.

def hack_highfreq_amp(X, max_amp = 1.5):
    '''Amplifies the higher frequency components of a chirp
    by multiplying it with a maximum amplitude value. 
    
    ACHTUNG: this ONLY works with high-->low frequency sweeps
    '''
    cosine_size = X.size*2
    full_window = signal.cosine(cosine_size)
    half_window = full_window[X.size:]
    # offset by 1 and set maximum value
    hack_amp_window = (half_window +1)*max_amp
    return(hack_amp_window)

# and now make all the 9 types of calls : 
start_f, end_f = 95000, 15000

all_waveforms = []
i = 0 

for call_type in ['linear','logarithmic','bidirectional']:
    for call_durn in [0.006, 0.012, 0.024]:
        t = np.linspace(0, call_durn, int(fs*call_durn))       
        this_waveform = make_chirp_waveform(t, start_f, end_f, call_type)
        # amplify the high frequency compoentnet with the hack window :
        amp_high_freqs = hack_highfreq_amp(this_waveform, 24.0)
        if call_type != 'bidirectional':
            this_waveform = this_waveform * amp_high_freqs
           
        else:
            half_waveform = int(this_waveform.size/2.0)
            half_amp_high_freqs = hack_highfreq_amp(this_waveform[:half_waveform], 24.0)
            this_waveform[:half_waveform] *= half_amp_high_freqs
            this_waveform[half_waveform:] *= half_amp_high_freqs[::-1]
            
        this_waveform *= 1/np.max(this_waveform)
        this_waveform *= 0.9 

        this_waveform *= signal.tukey(this_waveform.size, 0.15)
        num_pad_samples = abs(sync_winsamples - t.size)        
        waveform_w_silence = np.concatenate((np.zeros(num_pad_samples), this_waveform))
        all_waveforms.append(waveform_w_silence)
